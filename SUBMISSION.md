# 作业提交说明（可直接交）

## 作业题目

利用 AI 与开源社区工具，采用“复制粘贴 + 组合改造”方式完成一个带功能的网页项目，并提交到个人 GitHub 仓库。

## 我完成的项目

- 项目名称：淘趣集市（类淘宝页面）
- 类型：纯前端功能网页
- 访问入口：`index.html`

## 功能清单（对应课程要求）

- 跳转功能：首页、商品页、详情页、购物车页可互相跳转
- 响应式：在手机和桌面端都可正常显示
- 动态刷新：
  - 首页限时推荐每 8 秒自动刷新
  - 购物车数量和总价随操作实时变化
- 交互功能：
  - 商品搜索、分类筛选、排序
  - 加入购物车、数量增减、删除、清空
  - 购物车本地持久化

## AI + 社区工具使用说明

本作业按照课程“尽量不手写”思路完成，采用以下流程：

1. 用 AI 生成页面结构和交互逻辑草稿。
2. 参考社区常见电商页面结构（导航 + 筛选 + 商品卡片 + 购物车）进行组合调整。
3. 通过复制粘贴与小幅改造完成最终项目。
4. 本地运行验证后提交到 GitHub。

## 本地运行方式

```bash
python -m http.server 8080
```

打开：

```text
http://localhost:8080/index.html
```
